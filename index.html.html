Sinkronisasi Google Sheets
Backup dan sinkron data 2 arah dengan Google Spreadsheet
Konfigurasi Koneksi
● Belum terhubung
Cara Setup
1. Buka Google Sheets → Extensions → Apps Script
2. Paste isi file GoogleAppsScript.js, ganti token, Deploy → Web App → Anyone
3. Copy URL deployment, paste di bawah ini
Push ke Google Sheets

Kirim semua data lokal ke Spreadsheet. Data di Sheets akan ditimpa.

Pull dari Google Sheets

Ambil data dari Sheets. Data lokal akan digabung (merge).

Data lokal yang tidak ada di Sheets akan tetap dipertahankan.
